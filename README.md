# Sudoku

「数独」

（概要・テーマ）
この作品は株式会社ヘキサドライブで、プロジェクト「Habit」を開発する際に個人的に勉強する為に開発した作品です。
この作品の開発によって、MVPパターンやファサードパターンなどを学ぶことができました。

(開発時期・期間)
ヘキサドライブ時代の2025年8月～2025年9月(1ヶ月)

（開発ツール）
・ゲームエンジン：Unity(バージョン2022.3.62f2)
・言語：C# 
・開発環境：Visual Studio Code

（チームメンバー）
・プログラマー1人（担当）

（ゲームシステム）
3×3のグループ(ブロック)に区切られた 9×9の正方形の枠内に1～9までの数字を入れるパズルゲームです。

(操作方法)
・マウスでボタンをクリックして進めてください。

(起動方法)
「Sudoku(ビルド)」というフォルダーを開いてください。
その中にある、「Sudoku.exe」というアプリケーションをダブルクリックしてください。
